# 📱 QR Code Attendance System

## 🚀 Overview

The **QR Code Attendance System** is a smart and efficient solution designed to automate attendance tracking using QR codes. It eliminates manual entry, reduces errors, and saves time by enabling quick scanning and real-time data recording.

---

## 🎯 Features

* 📷 QR Code Generation for users/students
* ✅ Instant Attendance Marking via QR Scan
* ⏱️ Real-time Attendance Tracking
* 📊 Attendance Reports & History
* 🔐 Secure Authentication (Admin/User)
* 📱 Cross-platform support (Android / Web / Desktop)

## 📸 How It Works

1. Admin generates a unique QR code.
2. Users scan the QR code using the app.
3. Attendance is recorded instantly in the database.
4. Reports can be viewed anytime.
